# Synthesis

Council synthesis integrates role-specific outputs. It should not treat the
roles as votes because each role has a different lens and responsibility.

## Required Findings

1. **Agreements**: facts or recommendations supported across roles.
2. **Conflicts**: role priorities, evidence, or recommendations that conflict.
3. **Partial coverage**: important topics only some roles addressed.
4. **Unique insights**: valuable role-specific observations.
5. **Blind spots**: relevant questions or evidence nobody addressed.

## Final Answer

Lead with the strongest supported recommendation, preserve conflicts that affect
the decision, incorporate role-specific insights when justified, and explicitly
name blind spots that affect confidence. For code or artifact tasks, prefer
verification evidence over persuasive prose.
